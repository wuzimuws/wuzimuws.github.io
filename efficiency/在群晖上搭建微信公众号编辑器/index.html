在群晖上搭建「微信公众号编辑器」—— doocs/md: ✍ WeChat Markdown Editor

这是一款高度简洁的微信 Markdown 编辑器：支持 Markdown 语法、色盘取色、多图上传、一键下载文档、自定义 CSS 样式、一键重置等特性，将上述服务搭建在群晖的Docker上，在本地就能方便地使用免费的微信公众号编辑器，而且可以根据自己的需求进行自定义设置，终于不用再忍受收费插件的各种限制和广告了！

先使用ssh连接到群晖，安装md服务#

由于目前屏蔽的原因，无法使用群晖docker 中的注册表直接安装

  1. 打开群晖的SSH功能
  2. SSH连接到群晖并安装 如果直接用docker run -d -p 8080:80 doocs/md:latest提示无权限，则需要用sudo给权限，即sudo docker run -d -p 8080:80 doocs/md:latest。

设置Docker启动服务自动重启，大功告成#

登陆http://192.168.1.115:8080/即可使用免费的「微信公众号编辑器」啦 🎉🎉🎉！
