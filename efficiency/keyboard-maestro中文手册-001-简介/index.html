Keyboard Maestro 也许是适用于 Mac 的最强大的自动化工具。

最重要的是，任何人都可以简单上手。

Keyboard Maestro 不依赖于脚本语言，只需要设置好 「触发器 Trigger」 和「动作 Action」，即可制作出满足需求的宏。

逻辑简单🥳#

任务自动化的逻辑非常简单，只需要三步：

  • 拆解任务
  • 选择触发器（Trigger）
  • 设置动作（Action）

下一节，进入快速上手阶段，通过一个简单示例，来快速了解一下 Keyboard Maestro 的运行逻辑！
