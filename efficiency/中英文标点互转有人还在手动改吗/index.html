制作Keyboard Maestro宏#

在文字或代码处理过程中总会有中英文标点转换的问题，比如对于这样的JSON数据：

[
	{
		"name": “a”,
		"selected": “1”,
		“deadline”: “3”
	},
	{
		"name": “a”,
		"selected": “1”,
		“deadline”: “3”
	}
]

一旦其中出现中文标点，比如最后一行写成"deadline": "3“，那就惨了，找Bug找半天，可能都不知道问题出在哪，单靠肉眼也很难分辨。

要解决这种常发性的问题，就做个宏吧！

咱们的宏要实现的功能如下：

  • 如果标点中包含中文标点，则将全部标点修改为英文
  • 如果不包含则将全部标点修改为中文，实现标点的统一，同时可以实现中英文标点的互转

具体效果上视频吧！

这个宏很简单，也很实用，希望能帮助到大家！
