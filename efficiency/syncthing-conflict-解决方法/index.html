Fix sync conflicts#

修复同步冲突

  • On all devices, open the Syncthing web UI 在所有设备上，打开 Syncthing 网页界面

  • Go to Actions → Advanced → Folder → Select your Obsidian folder → Max Conflicts → Set to 0 转到操作 → 高级 → 文件夹 → 选择您的 Obsidian 文件夹 → 最大冲突 → 设置为 0

  • Repeat for each device 对每个设备重复此操作

  • This will make Syncthing just keep the latest copy (which from what I’ve seen is always the correct copy) 这将使 Syncthing 只保留最新的副本（据我所见，这始终是正确的副本）

If this is too much of a pain for you, consider Obsidian Sync - the devs made a great tool and support is always appreciated

如果这对你来说太麻烦，请考虑 Obsidian Sync - 开发人员制作了一款出色的工具，并且始终感谢他们的支持
