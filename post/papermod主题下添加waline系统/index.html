papermod主题下添加waline系统#

Papermod个性化配置#

为了避免破坏原有样式文件，将thems/PaperMod/layouts下的_default和partials文件夹均拷贝到主文件夹下的layouts文件夹中，在该文件夹下进行配置。 (img)

🔗Getting Started With Hugo

修改toc位置#

将thems/PaperMod/assets下的css文件夹拷贝到主文件夹下的assets文件夹中,在/common/post-single.css中修改 .toc { margin: 0 2px 40px 2px; border: 1px solid var(–border); background: var(–code-bg); border-radius: var(–radius); padding: 0.4em; position: fixed; top: 100px; right: 5%; width: 20%; max-height: 80vh; overflow-y: auto; } 要在手机端不显示 .toc，您可以使用媒体查询来达到这个效果。以下是一个示例，该示例假定手机屏幕的宽度小于或等于 768px（这是一个常用的断点，但您可以根据需要进行调整） @media (max-width: 768px) { .toc { display: none; } }

添加Waline评论系统到Papermod主题#

  • 在config.yml中打开comments开头： params: comments: true

  • 将 Waline 样式添加到头部: 打开你的 Hugo 项目中的 layouts/partials 文件夹下的 head.html 文件，然后在适当的位置（通常是其他样式表链接的下方）添加以下代码：

<link rel="stylesheet" href="https://unpkg.com/@waline/client@v2/dist/waline.css" />
  • 将 Waline 的客户端脚本和初始化代码添加到 comments.html:

在 layouts/partials 目录下创建或编辑 comments.html 文件，然后加入以下代码：

 <div id="waline"></div>
   <script type="module">
     import { init } from 'https://unpkg.com/@waline/client@v2/dist/waline.mjs';

     init({
       el: '#waline',
       serverURL: 'YOUR_WALINE_SERVER_URL',
     });
   </script>
