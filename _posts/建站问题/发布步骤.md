# 发布步骤
####发布的步骤
在mweb写完了之后，双击文档库，清理并重新生成网站，随后可通过两种方式进行发布：

1.在**终端**通过下面列出的命令进行push：

```git push -u origin master```

2.在GitHub desktop 的左侧变更中进行commit，然后push，具体参见[**发布过程中遇到的问题**](mweblib://15145271894707)

经验证，两种方式都行，就看命令行和app端哪个开着了～


```echo "# wuzimuws.github.io" >> README.md
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/wuzimuws/wuzimuws.github.io.git
git push -u origin master
```
